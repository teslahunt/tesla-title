> Get Tesla vehicles title based on option codes.

## Install

```bash
$ npm install tesla-title --save
```

## Usage

```js
const title = require('tesla-title')

title({ optionCodes: ['MDLS', 'BT85', 'DV4W'] }) // => 'Model S 85D'
```
